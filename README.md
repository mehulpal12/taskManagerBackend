# ⚙️ Team Task Manager – Backend

A scalable **Node.js + Express backend** powering a Team Task Manager application with authentication, RBAC, and task management.

---

## 🌐 Live API

👉 https://taskmanagerbackend-uc0i.onrender.com

---

## 🧩 Features

* 🔐 JWT Authentication (Signup/Login)
* 🛡️ Role-Based Access Control (Admin / Member)
* 📁 Project management
* 👥 Team member management
* 📌 Task assignment system
* 🔄 Task status updates
* 📊 Dashboard-ready data
* 🔒 Secure middleware validation

---

## 🏗️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (Atlas)
* **ODM:** Mongoose
* **Auth:** JWT
* **Architecture:** Controller → Service → Model
